<h1 align="center"> Simple HTTP, REST and SSE client library for Go </h1>

httpx-go offers improved performance, memory efficient client for Go. Is somewhat opinioted and highly configurable.

## Key Features
- Simple chainable methods
- Retry Mechanism
- Circuit Breaker
- Concurrent Safe
- Automatic Decompression
- Request Tracing
- CURL command generation
- Citcuit breaker condition
- Automatic marshal and unmarshal
